An ML-powered stock analysis and forecasting platform with an interactive Streamlit dashboard. Combines technical analysis, classification and regression models, and a strategy backtesting engine into a single, deployable application.

---

## What It Does

| Capability | Description |
|---|---|
| **Real-time Data** | Fetches historical OHLCV data for any ticker via `yfinance` (no API key required). |
| **Technical Indicators** | SMA, EMA, RSI, MACD, Bollinger Bands, ATR, momentum, volatility, and more — all computed in pure pandas. |
| **Direction Prediction** | Random Forest and Gradient Boosting classifiers predict next-day price direction (up/down). |
| **Price Forecasting** | Linear Regression and Random Forest regressors predict next-day closing price with confidence bands. |
| **Backtesting** | Compares an ML-driven long/cash strategy against buy-and-hold, reporting Sharpe ratio, max drawdown, and win rate. |
| **Walk-Forward Validation** | Expanding-window validation ensures models are evaluated on truly unseen future data. |

---

## Architecture

```
app.py  (Streamlit UI — 5 interactive tabs)
 │
 ├── src/data_fetcher.py        Yahoo Finance data retrieval + caching
 ├── src/feature_engineering.py  Technical indicators & ML feature builder
 ├── src/models.py               Classifiers, regressors, walk-forward CV
 ├── src/backtester.py           Strategy simulation engine
 └── src/utils.py                Colours, formatters, Plotly helpers
```

---

## Features

- **Bloomberg-terminal-inspired dark UI** with custom CSS
- **Candlestick charts** with toggleable SMA, EMA, Bollinger Band overlays
- **Confusion matrix & feature importance** visualisations
- **Predicted vs actual price** with +/- RMSE confidence band
- **Portfolio equity curve** — strategy vs buy-and-hold
- **Signal summary table** — bullish / bearish / neutral at a glance
- **Time-series-aware splitting** — no data leakage, no random shuffle
- Parquet-based data caching for fast repeat queries

## Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/eduardomoraga/stock-price-predictor.git
cd stock-price-predictor

# 2. Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate   # Linux/macOS
venv\Scripts\activate      # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Launch the app
streamlit run app.py
```

The app opens at **http://localhost:8501**. Enter any ticker (e.g. `AAPL`, `MSFT`, `TSLA`), pick a date range, choose your models, and explore.

---

## Deploy to Streamlit Cloud

1. Push this repo to GitHub.
2. Go to [share.streamlit.io](https://share.streamlit.io) and connect the repo.
3. Set the main file path to `app.py`.
4. Deploy — no secrets or environment variables needed.
